# 多贷款计算器

一个纯浏览器端运行的贷款还款计划计算器，支持同时管理多笔贷款，并可模拟提前还款、利率调整等事件对还款计划的影响。

## 核心特性

### 隐私保护 - 纯本地计算

**你的财务数据完全属于你自己。**

- **纯本地运行**：所有计算在浏览器本地完成，无任何网络请求
- **无需账号**：不需要注册、登录，无用户追踪
- **数据本地存储**：贷款数据仅保存在浏览器 localStorage 中
- **可离线使用**：断网状态下可正常使用所有功能

> 验证方式：打开浏览器开发者工具的 Network 面板，可见无任何数据上传。

### 功能特点

- **多贷款组合**：同时管理多笔贷款，自动合并展示总还款计划
- **还款方式**：支持等额本息、等额本金两种还款方式
- **事件模拟**：支持提前还款、利率调整等事件，实时计算对还款计划的影响
- **数据可视化**：清晰的还款计划表格，悬停可查看累计还款数据

## 技术栈

- React 19 + TypeScript
- Tailwind CSS
- Zustand（状态管理）
- Zod（数据校验）
- Vitest（单元测试）

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 运行测试
pnpm test
```

## 使用说明

1. **添加贷款**：点击左侧添加按钮，输入贷款金额、利率、期限等信息
2. **配置事件**：在贷款详情中添加提前还款、利率调整等事件
3. **查看计划**：右侧表格展示合并后的完整还款计划
